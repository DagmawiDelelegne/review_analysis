Users can paste multiple reviews on separate lines in the interface. Atfter running the program on the reviews a table showing the aspects and the number of times they were labled Positive, Negative, or Neutral will be shown.

# Tools used
- PyABSA
- Gradio
- Pandas
# Files
- sentimentClassifier.py (main program)
- requirements.txt
# setup
- Recommend using python 3.11
- create a virtual enviroment and install the requirements
- ```python -m pip install -r requirements.txt```
# Run 
- ```sentimentClassifier.py```
- open local gradio interface
- paste reviews in the text box and press submit
